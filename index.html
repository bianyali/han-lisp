  • Introduction

A dialect of Lisp

This is dialect of Lisp, writen in Racket.

In each s-exp symbol are writen in Chinese,so han-lisp means 汉-lisp.

  • Installation

    1. Install Racket
    2. Open han.rkt
    3. run racket
    4. type han-lisp in repl terminal

  • Demonstration

(设 (函数名 输入1 输入2))

(若 判断条件 程序分支1 分支2)

(若 真 1 2) (若 假 1 2)

(令 变量1 "你好，世界")
